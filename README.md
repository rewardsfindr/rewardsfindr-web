# RewardsFindr Web - Landing Page

Marketing website for RewardsFindr mobile app.

## 🌐 Live Site

https://rewardsfindr.com (deployed on Vercel)

## 📱 What is RewardsFindr?

RewardsFindr helps you maximize credit card rewards by automatically tracking your bank offers and showing you which card to use at any store.

## 🛠 Tech Stack

- React 18
- CSS (custom, no frameworks)
- Deployed on Vercel

## 🚀 Development

```bash
# Install dependencies
npm install

# Start dev server
npm start
# Visit http://localhost:3000

# Build for production
npm run build
```

## 📁 Project Structure

```
/src
  App.js          # Main landing page component
  App.css         # Landing page styles (green dollar theme)
  index.js        # React entry point
  index.css       # Global styles
/public
  index.html      # HTML template
  favicon.ico     # Site icon
```

## 🎨 Design

- **Color Theme:** US Dollar green (#059669, #047857)
- **Sections:** Hero, How It Works, Savings Stats, Features, FAQ, Footer
- **Responsive:** Mobile-friendly design
